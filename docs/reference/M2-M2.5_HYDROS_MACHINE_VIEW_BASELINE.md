# HydrOS Machine View — Visual Baseline Reference

> **[M1.5 STALENESS NOTE — 2026-04-10]**
> This baseline image was created before the M1.5 physics correction report. It contains two known schematic errors that will be corrected in M3:
> 1. **Electrode geometry**: image shows axial field lines; correct geometry is radial (outer cylindrical wall as collector, field normal to mesh surface). See `docs/research/2026-04-10-physics-correction-report.md` Issue 1.
> 2. **Collection tubes**: image may show symmetric tube routing; correct design is downward-only gravity-fed tubes (dense-phase only, ρ > 1.0 g/cm³). See `docs/context/06_LOCKED_SYSTEM_CONSTRAINTS.md` §E.
>
> This image remains the visual identity anchor. It must NOT be used as a schematic reference for electrode placement or particle collection topology until updated in M3.
