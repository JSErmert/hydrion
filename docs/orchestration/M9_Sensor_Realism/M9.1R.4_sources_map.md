## 5. Risk Assessment

### 5.1 Weakest grounded parameter

**`dp_fouling_gain = 0.02`** at MEDIUM confidence. This is the only calibration-target parameter without a direct quantitative source for its exact magnitude. It is bounded by three independent installation-practice sources (S7, S8, S9) to be much smaller than the 0.5 placeholder, and the direction of calibration (large reduction) is certain. The exact value (0.02 vs 0.01 vs 0.05) is within a factor of ~2.5 uncertainty.

**Risk if wrong by 2.5×:**
- If 0.01: at full fouling (`mesh_loading_avg=1.0`), sensor reads 0.01 kPa high — undetectable against 0.25 kPa white noise
- If 0.05: sensor reads 0.05 kPa high — still small against 0.25 kPa white noise
- In either case, the behavioral impact on the policy is negligible at episode scale. The parameter is retained for structural completeness (multi-hour future simulation), not for intra-episode effect.

**Assessment:** Low risk. The uncertainty in `dp_fouling_gain` does not materially affect the M9 calibrated training or evaluation. It is the weakest parameter by provenance quality but the least impactful by behavioral effect.

### 5.2 Most critical assumptions

| Assumption | Used in | Impact if wrong |
|-----------|---------|-----------------|
| **HydrOS dt = 0.1 s per step** | dp_drift_rate conversion from per-year to per-step | If dt = 0.05 s, drift per step halves to 6.3e-7. If dt = 1.0 s, drift per step increases to 1.27e-5. In all cases drift remains negligible against white noise. **Low sensitivity.** |
| **MEMS 0.5% FS/year bound (conservative)** | dp_drift_rate, dp_drift_max | If the actual device is industrial-grade (0.2%/year), calibrated drift shrinks further. If the device is lower-quality (1%/year), calibrated drift doubles — still 200× smaller than placeholder. **Low sensitivity to device-class grade.** |
| **±2% reading accuracy ≈ 1-sigma** | turbidity_noise_std, scatter_noise_std | If ±2% is 2-sigma, noise_std should be 0.01 (same as current placeholder). If 3-sigma, noise_std ≈ 0.007. The worst case is that the calibrated value is the same as the placeholder — no harm done. **Low sensitivity.** |
| **Direct-mount transducer (no impulse lines)** | dp_fouling_gain | If impulse lines are used, gain could be 5–10× larger (0.1–0.2). Still much smaller than the 0.5 placeholder. **Low sensitivity.** |

**No assumption has high sensitivity.** All calibrated values are robust to the identified assumption uncertainties — the order-of-magnitude corrections (dp_drift 400× reduction; dp_fouling 25× reduction) dominate any within-range variation from assumption uncertainty.
